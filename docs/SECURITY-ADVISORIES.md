# Banco de Advisories de Segurança

Não há vulnerabilidades confirmadas ou advisories publicados para a versão `0.1.0`.

Cada advisory futuro deve usar o identificador `ORBITA-HRM-YYYY-NNN`, conter versões afetadas e corrigidas, CWE, CVSS, impacto, mitigação, datas e link público de divulgação após a correção.

## Modelo

| Campo | Valor |
| --- | --- |
| Identificador interno | ORBITA-HRM-YYYY-NNN |
| CWE | CWE-XXX |
| CVSS | 0.0 |
| Versões afetadas | a definir |
| Correção / mitigação | a definir |
| Status | em avaliação |
