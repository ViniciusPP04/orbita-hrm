# Política de Segurança

## Versões suportadas

| Versão | Suportada |
| --- | --- |
| 0.2.x | Sim |
| 0.1.x | Não |

## Reporte responsável

Não publique detalhes de uma possível vulnerabilidade antes da coordenação com o fornecedor. Envie uma descrição, passos de reprodução, impacto, versões afetadas e evidências para `security@orbita.local`.

O time confirma o recebimento em até 3 dias úteis, avalia o relato e publica um advisory quando houver impacto confirmado e uma correção ou mitigação disponível.

## Escopo

Inclui a API local, autenticação, autorização, gestão de documentos e dependências da aplicação. Este MVP não deve ser exposto à internet nem usado com dados pessoais reais.
